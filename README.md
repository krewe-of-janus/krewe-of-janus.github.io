# krewe-of-janus.github.io

## Overview
Single-page static site for Krewe of Janus with a centered logo, title/subhead, contact link, and fixed Mardi Gras accent bars on the left.

## Structure
- [index.html](index.html): HTML5 layout with the left-side bars, title block, logo, contact link, and footer copyright.
- [styles.css](styles.css): Minimalist styling, Mardi Gras color palette, responsive spacing, and typography.
- Janus Logo.png: Main logo image referenced by the page.

## Layout Notes
- Left sidebars are fixed and span the full height of the viewport.
- Content is centered vertically and horizontally within the remaining space.
- The copyright line stays pinned to the bottom of the visible screen.
