# krewe-of-janus.github.io

## Overview
Static multi-page site for Krewe of Janus with a centered logo, title/subhead, social links, and fixed Mardi Gras accent bars on the left.

## Structure
- [index.html](index.html): Main landing page with navigation to subpages, social icons, and footer.
- [2026-court.html](2026-court.html): 2026 Court page with headshots and a 2x2 grid of court members.
- [queens-of-janus.html](queens-of-janus.html): Queens list displayed in two columns.
- [history.html](history.html): Historical narrative with thumbnail photo grids under each paragraph.
- [styles.css](styles.css): Shared styling, layout, typography, and responsive behavior.
- Janus Logo.png: Main logo image used across pages.
- images/2026-court/: Headshots for the 2026 court page.
- images/history/: Historical photo thumbnails.
- resize-images.zsh: Script for resizing general images.
- resize-history-images.zsh: Script to resize and strip metadata from history images.
- rename-history-images.zsh: Script to anonymize history image filenames.

## Layout Notes
- Left sidebars are fixed and span the full height of the viewport.
- Content remains centered and responsive on all pages.
